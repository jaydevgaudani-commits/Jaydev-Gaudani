# SPEC.md — Project Specification

> **Status**: `FINALIZED`

## Vision
A high-end personal brand portfolio for an AI automation expert and web developer. The system transitions the "Terminal Curator" aesthetic into a "Liquid Glass" interface, using WebGL-driven liquid physics to create a premium, tactile experience that showcases automation results.

## Goals
1. Establish authority in AI automation and web development through a professional, technical aesthetic.
2. Showcase real-world impact by highlighting metrics like "98% EFFICIENCY" and custom projects like "CLINIC_OS_V1", "Custom Web Platforms", and "n8n Logic Flows".
3. Provide a high-engagement, interactive experience using WebGL distortion.
4. Maintain a strict "No-Line" rule to define structure through tonal shifts.

## Non-Goals (Out of Scope)
- Personal blog or non-professional content.
- Detailed technical tutorials (for this version).

## Users
- **Primary:** Business owners and decision-makers looking to automate systems or build custom web architectures.

## Constraints
- **Design System:** "Terminal Curator" / "Minimalist Cyberpunk".
- **Glass Spec:** "Liquid Glass" (40px blur, 10% opacity border).
- **Layout:** Bento Grid focus with "No-Line" boundaries (#0E0E0E vs #131313).
- **Visuals:** Abstract "Terrain_04" style only (no stock images).
- **Physics:** WebGL overlay cursor distortion.

## Success Criteria
- [ ] Hero text effectively communicates "Automating System Logic".
- [ ] Project cards correctly showcase: 'Clinic Automation Systems', 'Custom Web Platforms', and 'n8n Logic Flows'.
- [ ] WebGL liquid physics implemented on the grid.
- [ ] Bento cells use "Liquid Glass" styling.
- [ ] "No-Line" rule strictly followed.
